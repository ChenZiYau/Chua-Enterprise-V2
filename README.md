# Chua Enterprise V2

Property management admin dashboard — design/layout phase.

Stack: Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4.

## Run

```bash
npm install
npm run dev
```

Then open http://localhost:3000 (auto-redirects to `/admin`).

## Status

Visual shell only — no data, no auth, no backend yet.
